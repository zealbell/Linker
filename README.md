# Linker

This is a premium personal website designed for a creative professional.

## Setup
1.  Open `index.html` in your browser to view the site.
2.  To deploy, push this folder to a GitHub repository and enable GitHub Pages.

## Structure
-   `index.html`: Main content.
-   `css/style.css`: Styles.
-   `js/app.js`: Logic.
-   `assets/`: Images and resume.
